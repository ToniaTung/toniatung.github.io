## Overview
Designed and implemented an autonomous mobile vacuum robot capable of obstacle avoidance,
wall-following navigation, and debris collection.  
The system integrates mechanical design, embedded control, sensor fusion, and vision-based detection.

---

## System Architecture
- Arduino-based motor and PID control
- Raspberry Pi for vision-based color detection
- Ultrasonic sensors for distance measurement and obstacle avoidance
- H-bridge motor driver and DC motors
- Custom-designed vacuum module and chassis

---

## Mechanical Design
- Designed vacuum fan, dust container, and intake geometry
- Optimized airflow performance using Autodesk CFD simulation
- Redesigned chassis to improve stability, wiring layout, and component accessibility
- Fabricated structural and vacuum components using 3D printing

---

## Control & Algorithms
- Implemented PID-based wall-following control
- Designed distance and orientation error metrics using dual ultrasonic sensors
- Tuned control gains to achieve stable navigation without predefined paths
- Limited wheel speed to prevent oscillations and instability

---

## Vision & Embedded Systems
- Developed Raspberry Pi pipeline for real-time color detection
- Implemented inter-processor communication between Raspberry Pi and Arduino
- Designed fault-handling strategies to address communication overload issues

---

## Competition & Results
- Successfully demonstrated autonomous navigation in final competition
- Achieved reliable debris collection and wall-following behavior
- Only team to deploy a fully autonomous control strategy
- Identified hardware reliability and system integration as key areas for future improvement
